Ben 10 API Documentation

Welcome to the Ben 10 API! This API provides information about Ben 10 aliens and characters.

Base URL

The base URL for the API is: https://ben10restapi-fl6k4l0g.b4a.run

Endpoints

Aliens

Retrieve All Aliens

GET /api/alien

Returns a list of all aliens.

Example Response
[
    {
        "id": 1,
        "name": "Heatblast",
        "species": "Pyronite",
        "planet": "Pyros",
        "abilities": ["Fire manipulation", "Heat resistance"],
        "image": "https://ben10restapi-fl6k4l0g.b4a.run/images/Heatblast.png"
    },
    ...
]

Retrieve a Random Alien

GET /api/alien/random

Returns a random alien.

Example Response
{
    "id": 3,
    "name": "XLR8",
    "species": "Kineceleran",
    "planet": "Kinet",
    "abilities": ["Super speed", "Enhanced reflexes"],
    "image": "https://ben10restapi-fl6k4l0g.b4a.run/images/XLR8.png"
}

Retrieve an Alien by ID

GET /api/alien/id/:id

Returns an alien by its ID.

Example Response
{
    "id": 1,
    "name": "Heatblast",
    "species": "Pyronite",
    "planet": "Pyros",
    "abilities": ["Fire manipulation", "Heat resistance"],
    "image": "https://ben10restapi-fl6k4l0g.b4a.run/images/Heatblast.png"
}

Characters

Retrieve All Characters

GET /api/character

Returns a list of all characters.

Example Response
[
    {
        "id": 1,
        "name": "Ben Tennyson",
        "species": "Human",
        "role": "Hero",
        "abilities": ["Omnitrix wielder"],
        "description": "The main protagonist of the series.",
        "image": "https://ben10restapi-fl6k4l0g.b4a.run/images/BenTennyson.png"
    },
    ...
]

Retrieve a Random Character

GET /api/character/random

Returns a random character.

Example Response
{
    "id": 2,
    "name": "Gwen Tennyson",
    "species": "Human",
    "role": "Hero",
    "abilities": ["Magic", "Martial arts"],
    "description": "Ben's cousin and a skilled magic user.",
    "image": "https://ben10restapi-fl6k4l0g.b4a.run/images/GwenTennyson.png"
}

Retrieve a Character by ID

GET /api/character/id/:id

Returns a character by its ID.

Example Response
{
    "id": 1,
    "name": "Ben Tennyson",
    "species": "Human",
    "role": "Hero",
    "abilities": ["Omnitrix wielder"],
    "description": "The main protagonist of the series.",
